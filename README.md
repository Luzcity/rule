# rule

FiveM向け日本語RPサーバーのルールを、GitHub Pagesでそのまま公開できる静的サイトとしてまとめた初版です。

## Files

- `index.html`: トップページ
- `styles.css`: 共通スタイル
- `pages/basic-rules.html`: 基本ルール
- `pages/crime-rules.html`: 犯罪・抗争ルール
- `pages/public-service-rules.html`: 警察・EMS方針
- `pages/operations-policy.html`: 運営対応ポリシー

## Publish

GitHub Pages を `Deploy from a branch` に設定し、`main` ブランチの `/ (root)` を公開対象にすると、そのまま表示できます。
